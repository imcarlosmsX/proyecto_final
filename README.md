# proyecto_final

Pasos preliminares
Crear base de datos llamada "bakery_quilla" desde HeidiSQL
en consola:
pip install -r requirements.txt
py manage.py makemigrations ventas
py manage.py migrate


Para el backend
en consola:
py manage.py runserver


Para el frontend
cd client
npm run dev

Requirimientos adicionales:
node.js
MariaDB
HeidiSQL
python
